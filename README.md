*Please read before installation*

To install and initialise this project, you must clone the repo and then install docker and python (install python in the backend folder, set up a new virtual environment within it).


1. Run `` git clone ``
2. Install docker ``sudo apt update`` then ``sudo apt install docker-compose-plugin`` and ``docker compose version``
3. To start the docker container, in the telle3118 folder, run ``docker compose up`` (alternatively try ``docker-compose up`` if it does not work)
